# 核心问题 · 选择性必修3 生物技术与工程

> 本模块聚焦于将基础生物学知识转化为实际应用的技术与工程。在这里，我们从“认识生命”迈向“改造与利用生命”。

内容涵盖了从传统发酵工程到现代基因工程的各项关键技术，探讨它们如何深刻地改变医药、农业、环保等领域，并审视其带来的生物安全与伦理挑战。
