# 核心问题 · 必修2 遗传与进化

> 本模块深入探讨了生物学的两大核心主题：遗传，作为生命延续与传承的基石；进化，作为生命波澜壮阔、多样纷呈的宏伟史诗。

通过对经典遗传定律、基因的物质基础及其表达的梳理，我们将揭示生命性状代代相传的奥秘，并在此基础上，探寻生物种群演化的内在规律与外在驱动力。
