# 选择性必修二：生物与环境 · 人教版

本模块将我们的视野从生物个体内部拓展到外部环境，研究生命与环境之间的相互关系，即生态学。

内容从种群的特征和数量动态入手，逐步扩展到由多个种群构成的生物群落及其演替规律，最终聚焦于生物群落与无机环境共同构成的生态系统，探讨其结构、功能和稳定性。本模块还特别关注人类活动对环境的影响，旨在培养学生的环保意识和社会责任感。
